# My profile page

This portfolio is made with [Create React](https://github.com/facebook/create-react-app) App and [TypeScript](https://github.com/Microsoft/TypeScript).

## Easy Start

### local

- Node.js 14.16.1
- yarn 1.22.10

```
yarn install
yarn start
```

### docker

```
docker-compose up
```

## Deploy Step(local only)

```
yarn build
yarn deploy
```
